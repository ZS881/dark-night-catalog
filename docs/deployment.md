# 构建与部署说明

## 本地构建

```bash
cd app
pnpm install --frozen-lockfile
pnpm build
```

构建产物位于 `app/dist/`，可部署至任何支持静态文件托管的平台。

## 环境要求

- Node.js 22 或兼容的现代 LTS 版本
- pnpm 10

## GitHub Actions

仓库包含 `Frontend Quality Gate`：

- 对 `main` 的提交执行依赖锁定安装和生产构建。
- 对 Pull Request 执行同样的构建验证。
- CI 不接触用户上传的图片，也不需要生产密钥。

## 托管原则

本项目当前是纯静态 React 原型。选择托管时应确认：

1. 所在地区的移动网络可访问性。
2. 自定义域名、HTTPS 与必要的备案/合规要求。
3. 未来接入图像生成服务时的密钥隔离、速率限制和内容审核。
4. 不将图片上传到无明确告知、删除机制和访问控制的第三方服务。
