# 产品与体验系统

## 一句话定义

暗夜图鉴相机是一个将“自拍 / 物品照片”转化为平行异界观测入口的互动叙事产品。用户不会被评分或丑化；被看见的是一个与照片建立松散视觉联系、却拥有独立生活与职业的虚构人型居民。

## 目标体验

用户应在 3 分钟内完成一次完整体验：

1. 理解上传边界并完成同意。
2. 提供一张图片，选择想要靠近的异变倾向。
3. 获得一张有名字、职业、禁忌和来历的图鉴记录。
4. 自愿进入情境问卷，获得“平行身份”的娱乐性投影。
5. 在夜班章节中做出连续选择，获得可解释的存活或失落结局。
6. 将道具、关系线和下一夜线索收入本地档案馆。

## 体验层级

| 层级 | 用户问题 | 对应模块 |
| --- | --- | --- |
| 观测层 | “我在另一个世界会看见谁？” | 同意、锚点、异变倾向、图鉴卡 |
| 投影层 | “我会以什么身份出现？” | 16 题情境问卷、人格投影、深度报告 |
| 行动层 | “我能否在那个世界活过一夜？” | 四幕夜班、规则、选择、结局 |
| 收藏层 | “我还发现了什么？” | 世界线档案、道具、关系、残响追踪 |

## 不可妥协的边界

- 上传图片只服务于视觉锚定，不用于评价、诊断、识别或心理推断。
- 问卷为可选的娱乐玩法；MBTI 叙事不是科学测评、脑科学模型或固定定义。
- 默认不产生针对真人的公开评论区和外貌比较机制。
- 图像、档案与结局的恐怖感必须停留在虚构世界中。

## 成功指标（原型阶段）

| 指标 | 目标 | 观察方法 |
| --- | --- | --- |
| 首次完成率 | 5–8 人测试中 ≥ 70% 完成一条图鉴流程 | 观察 + 访谈 |
| 问卷进入率 | 完成图鉴后 ≥ 40% 主动进入问卷 | 本地事件计数 |
| 夜班完成率 | 启动夜班后 ≥ 60% 抵达一个结局 | 本地事件计数 |
| 误解率 | 不将产品理解为“外貌羞辱工具” | 访谈追问 |
| 分享意愿 | 至少半数测试者愿意保存纯图或分享图鉴 | 访谈 + 导出事件 |
