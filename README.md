# 暗夜图鉴相机

一个浏览器端的互动叙事原型：用户在明确同意后上传自己的照片作为“观测锚点”，完成可选的情境问卷，并进入虚构异界居民的图鉴、人格投影与分支夜班剧情。

## 项目结构

- `app/`：Vite + React 前端工程。
- `assets/`：原始氛围素材备份。

## 本地运行

```bash
cd app
pnpm install
pnpm dev
```

## 构建

```bash
cd app
pnpm build
```

## 内容边界

- 照片仅作为用户自愿提供的视觉锚点，不用于推断人格、身份、外貌价值或健康状态。
- 图鉴、问卷结果与剧情均为虚构娱乐内容，不构成心理评估或现实判断。
- 上传图片仅在当前浏览器会话中使用；原型不将图片上传至服务器。
